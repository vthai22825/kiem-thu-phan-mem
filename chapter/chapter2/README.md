# Chapter 02 - Unit Testing with JUnit

## Mục tiêu
Viết chương trình phân tích điểm học sinh và kiểm thử bằng JUnit.

## Chức năng
- Đếm số học sinh giỏi (>= 8.0)
- Tính điểm trung bình hợp lệ (0-10)

## Điều kiện kiểm tra
- Điểm < 0 hoặc > 10 sẽ bị loại
- Danh sách rỗng trả về 0

## Kiểm thử
Sử dụng JUnit 5 để test:
- Trường hợp bình thường
- Danh sách rỗng
- Có điểm không hợp lệ
